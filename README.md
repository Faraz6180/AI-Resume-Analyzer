# 🚀 AI Resume Analyzer for GCC Jobs

An AI-powered SaaS application that analyzes resumes and provides ATS score, job match insights, and improvement suggestions tailored for UAE 🇦🇪, Saudi Arabia 🇸🇦, and Qatar 🇶🇦 job markets.

## 🔥 Features

* 📄 Upload Resume (PDF)
* 📊 ATS Score Analysis (with visual progress)
* 🎯 Job Match Score (GCC Market)
* 🧠 AI-powered Resume Feedback
* 📥 Downloadable Report
* 🌍 GCC-focused job recommendations

---

## 🛠 Tech Stack

* Python
* Streamlit
* Groq (LLM API)
* PyPDF

---

## 🧠 How It Works

1. Upload your resume
2. App extracts text from PDF
3. AI analyzes resume using LLM
4. Generates:

   * ATS Score
   * Job Match Score
   * Skills & improvements
5. Displays structured insights
